# Extractor Tools

Small input-preparation tools for turning outside material into files the Ontology Machine can ingest.

## Tools

- `Article Archive Extractor` extracts article URLs into Markdown files.
- `YouTube Transcript Extractor` extracts available YouTube subtitles into Markdown files.
- `Audio Transcription Extractor` transcribes local audio/video files into Markdown files through the OpenAI Audio Transcriptions API.

Each tool has its own start BAT and README inside its folder. Keep the tool folders inside `Extractor_Tools` so the launchers can find the bundled Ontology Machine runtimes.
