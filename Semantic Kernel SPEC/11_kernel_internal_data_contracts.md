## Enum Sets

- `DatabaseEmptiness`: `empty`, `filled`, `unknown`.
- `SemanticReleaseState`: `no_semantic_release`, `semantic_release_incomplete`, `semantic_release_complete_not_active`, `semantic_release_active`, `unknown`.
- `AttachPointerOwner`: `kernel_held`, `pipeline_held`.
- `LockType`: `active_run`, `workspace`, `database`, `release_attach_activation`, `merge`, `rebuild_overwrite`.
- `LockStatus`: `active`, `released`, `failed`, `pending_resume`, `expired`.
- `InteractionKind`: `input`, `selection`, `confirmation`, `notice`, `recovery`.
- `DialogType`: `folder_picker`, `folder_create_picker`, `text_input`, `database_path_picker`, `active_database_choice`, `database_list_picker`, `input_presence_confirmation`, `update_mode_choice`, `generic_confirmation`, `blocker_notice`, `progress_notice`, `recovery_dialog`, `support_bundle_notice`.
- `AnalysisScope`: `sample_set`.
- `RiskClass`: `read_only`, `non_destructive`, `long_running`, `destructive`, `support`.

- `{{kernel_analyze_sample_inputs_json}}` feeds `analyze_samples`.

- `{{user_report_samples_seed_json}}` feeds `user_report_samples`.

## Removed Scope

The registry contains no database-analysis contracts, no comparison contracts, no existing-release mutation update-state contracts, no cleanup/reimport plans and no reimport continuation request. Frozen artifacts that still show historical labels are compatibility records, not active schema authority.
